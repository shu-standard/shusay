## ☕ About

はじめまして、shu-star（しゅうすたー）です。

季節の移ろいや、ふとした会話、ページをめくる音、
日常のなかにある「ちょっといいな」と思える瞬間を集めるのが好きです。

書くことは、自分の心を整理する作業でもあり、
誰かの一日にそっと寄り添うものでもあると思っています。
飾らない言葉で、等身大のエッセイを綴っていきます。
