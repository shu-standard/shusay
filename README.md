## ☀️ About

はじめまして、shu-star（しゅうすたー）です。

走ることと、書くこと。
ふたつの習慣のなかで、自分のペースを少しずつ見つけてきました。
横浜マラソンに毎年挑戦しながら、朝の時間を大切にする仲間と
**「ジブン株式会社朝活部」** を運営しています。

派手な記録でも、特別なノウハウでもありません。
「無理なく、でも続けていく」その過程を、noteに残しています。
